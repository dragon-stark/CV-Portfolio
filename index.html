Installation

To install Bumblebee on your web server, download the files from Themeforest and unzip them to the local disk on your computer. Inside the extracted folder you will see two folders named Documentation and HTML

HTML folder structure is as follows:

  • assets - contains all assets ( includes js, css and fonts )
  • index-1.html - Parallax Demo
  • index-2.html - Parallax Slider Demo
  • index-3.html - Youtube Demo
  • index-4.html - Liquid Distortion Demo
  • index-5.html - Dust Particle
  • index-6.html - Default Particle
  • index-7.html - Shape Particle
  • index-8.html - Snow Particle

Using an FTP Client (such as Filezilla or Total Commander), you will need to upload all of the contents of the HTML folder to your server. And other files can be used according to your preferences.

HTML Structure

This template is based on a responsive and full screen layout. And it's built using HTML5 standards.

index-*.html

From beginning of the <body> tag till the ending </body> tag, all the codes and texts are used for the template structure. You can change any text from the template by navigating through the index.html file and localizing each line of text you want to change.

Like index.html, you can change any page included with this template starting from <body> tag till ending </body> tag.

Change Placeholder Images

When you download the template files you will see we used placeholder images instead of those originals we used in our preview link. To change with your desired images replace the placehodlder images.

CSS Files

All the css files are located in the assets/css folder of the template.
bumblebee-light.css - Main Stylesheet for Light style template.
bumblebee-dark.css - Main Stylesheet for dark style template.
custom.css - Your own custom stylesheet. If you would like to edit a specific section of the template, simply find the appropriate labels and add respective styles there.

JavaScript Files

All vendor, 3rd party plugin and our custom JavaScript files are located in the js folder.
custom.js - Our custom JavaScript file. You can add your custom js codes there if you need to.
